# Arcade Vault — Gamepad MK-II

Componente de gamepad neón listo para usar, extraído del portal Arcade Vault.

## Archivos

- **`gamepad.html`** — Página standalone con el gamepad. Soporta teclado (flechas/WASD, Z/J = A, X/K = B) y eventos de puntero (mouse + táctil).
- **`gamepad-neon.png`** — Captura del componente

## Uso

Abre `gamepad.html` directamente en cualquier navegador moderno. Para incrustarlo en otro proyecto, copia el contenido del bloque `<style>` y el HTML dentro de `.page` — no requiere librerías externas (solo las fuentes de Google Fonts).
